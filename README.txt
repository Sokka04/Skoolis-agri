Projet Skoolis-AGRI
===================
1. Importer `skoolis_agri.sql` dans votre serveur MySQL (phpMyAdmin ou CLI).
2. Placer le dossier `php_app` dans votre serveur web local (ex: `C:/xampp/htdocs/`).
3. Modifier si besoin les accès dans `php_app/db.php` (user: root, pass vide par défaut).
4. Accéder à http://localhost/php_app/ dans votre navigateur.
